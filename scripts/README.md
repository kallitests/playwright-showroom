# 🛠️ Admin Scripts

Standalone Python utilities for Git workflow management, GitHub authentication, and environment validation.

Each script lives in its own subfolder with its own README. All scripts are **standard library only** unless noted — no pip install required to run them.

---

## Scripts

| Script | Purpose | AI-powered |
|---|---|---|
| [`git-commit-push/`](./git-commit-push/) | Hardened `add → commit → push` with pre-flight checks and secret detection | No |
| [`git-commit-push-ai/`](./git-commit-push-ai/) | Same as above but commit message is generated by Claude or Ollama from the staged diff | **Yes** (LangChain) |
| [`update-github-username/`](./update-github-username/) | Bulk-update GitHub remote URLs across all local repos after a username change | No |
| [`env-check/`](./env-check/) | Pre-launch validator — checks `.env`, packages, network, Docker, Playwright browsers | No |
| [`github-auth-setup/`](./github-auth-setup/) | Interactive wizard to fix SSH / PAT GitHub authentication errors | No |

---

## Quick start

```bash
# Secure commit & push (manual message)
python scripts/git-commit-push/git_commit_push.py

# Secure commit & push (AI-generated message via Ollama)
python scripts/git-commit-push-ai/git_commit_push_ai.py

# AI message via Claude instead
python scripts/git-commit-push-ai/git_commit_push_ai.py --llm-provider anthropic

# Validate your environment before running tests
python scripts/env-check/env_check.py

# Fix GitHub auth errors interactively
python scripts/github-auth-setup/github_auth_setup.py

# Update remote URLs after a GitHub username change
python scripts/update-github-username/update_github_username.py old_username new_username
```
